# SQL Sales Analysis Project

## Overview
This project analyzes sales data using SQL to extract business insights such as total sales, customer behavior, and category performance.

## Tools Used
- SQL (SQLite)
- CSV Dataset

## Key Questions Answered
- What are total sales and orders?
- Which categories generate the most revenue?
- Who are the top customers?
- How do sales trend over time?

## Insights
- Identified top-performing categories
- Found highest-value customers
- Observed monthly sales trends
